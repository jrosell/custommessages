# PrestaShop Custom Messages

A simple, lightweight module that adds a multilingual custom message in the displayNav1 hook for PrestaShop multistore setups.

## Features
- Shows a message with the selected messages.
- Free and open source.


##  Manual installation
1. Upload the `custommessages` folder to your `/modules/` directory.
2. Go to Module Manager in your Back Office.
3. Search for "Custom Messages" and install it.
4. You can customize the messages at the selected stores. It will appear in the top navigation (via `displayNav1` hook).

## Compatibility
- Works on PrestaShop 9.
- Works with Hummingbird and classic themes.
